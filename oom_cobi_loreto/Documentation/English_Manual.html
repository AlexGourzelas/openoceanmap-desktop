Conservation/Scientist Attributes

Interviews in this user group are typically done in addition to that of other user groups and is simply a way to describe briefly any other forms of income a person might have. Interviewing in this user group will produce a comma-separated value (CSV) text file instead of a shapefile since no shapes need to be captured.
